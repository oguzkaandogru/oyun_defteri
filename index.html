Ayarlar

Arama yapabilmek için ücretsiz bir RAWG API anahtarı gerekir. rawg.io/apidocs adresinden "Get API Key" ile kayıt olup anahtarını al, buraya bir kez yapıştır — cihazında saklanır, kimseye gönderilmez.

Tehlikeli bölge: aşağıdaki buton tüm listeni ve işaretlerini kalıcı olarak siler. Önce ⇩ ile yedek almanı öneririm.

GitHub yedekleme

İstersen listen GitHub deponda veri.json olarak da saklansın; böylece telefon ve bilgisayarda aynı listeyi görürsün, cihaz verisi silinse bile hiçbir şey kaybolmaz. Depo adını ve "fine-grained" erişim token'ını gir (token'ı yalnızca bu depoya, Contents: Read and write iznine ver).
